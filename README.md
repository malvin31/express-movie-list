# Express-Movie-List
A simple web application for movie fanatic by express 

## Features
+ listing movies from movie api
+ searching movies by title

### Searching 
+ type the title of movies in search bar

## Development Tools
+ Node.js v18.16.0
+ Express v4.18.2
+ Express-Handlebars v7.0.7
+ Bootstrap v5.1.3
+ popper.js v2.9.1
